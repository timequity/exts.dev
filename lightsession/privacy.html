LightSession Privacy Policy

Last updated: December 2025

Overview

LightSession is a browser extension that fixes ChatGPT UI lag by trimming old messages from the DOM. This policy explains what data the extension accesses and how it's used.

Data Collection

We collect nothing. LightSession does not collect, store, or transmit any personal data, browsing history, conversation content, or usage statistics to external servers.

Permissions Explained

LightSession requires the following browser permissions:

  • storage — To save your extension settings (enabled state, message count) locally on your device. This data never leaves your browser.
  • Host permissions (chat.openai.com, chatgpt.com) — To inject the content script that performs DOM trimming. The extension only runs on ChatGPT pages.

Local Storage Only

All your data is stored using browser.storage.local, which means:

  • Data is stored locally in your browser
  • Data does not sync across devices (by design — for privacy)
  • We have no access to this data — it's yours alone

Network Activity

LightSession makes zero network requests. The extension works entirely offline. We don't have servers, analytics, or any backend infrastructure.

Conversation Privacy

LightSession only trims the DOM (what the browser renders). It does not:

  • Read or store your conversation content
  • Send any data to external servers
  • Modify your actual conversation on OpenAI's servers

Your full conversation remains intact on OpenAI's side and can be restored by refreshing the page.

What We Don't Do

  • Track your browsing history
  • Collect analytics or usage data
  • Read your ChatGPT conversations
  • Send data to external servers
  • Show ads
  • Sell data (we have none to sell)
  • Require account creation

Open Source

LightSession is fully open source. You can audit the code yourself on GitHub to verify these claims.

Changes

If we ever change this policy, we'll update this page. Given our commitment to privacy, significant changes are unlikely.
